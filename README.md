# Tic Tac Toe Game in C

A simple two-player Tic Tac Toe game implemented in **C**. Players take turns placing X and O on a 3x3 grid. The game logs every move to a file and saves the final board and result to another file.

---

## 🎮 Game Overview

- **Objective**: Get three of your symbols (X or O) in a row, column, or diagonal.
- **Players**: 2 players (Player 1 = X, Player 2 = O)
- **Board**: 3x3 grid (positions 1-9)

---

## ⚙️ Features

- Two-player interactive gameplay
- Input validation (prevents invalid moves)
- Move logging to `game_log.txt`
- Saves final board & result to `game_result.txt`
- Displays board after each move
